How it works

Experiment with patterns and see visualisations evolve, sketched by your Polarbot. Here is how it works.

A Diagram of sorts

A schematic thing that shows the setup of a project.

Every visualisation the Polarbot sketches is 'project'. It is made up of a data source, to which a pattern is applied and a bot setup, so that the Bot knows how large it can draw. A pattern maker lets Users compose visualisations before sending them to polarbots to sketch.

Data Sources are open data sets like tidal information, weather, temperature or energy readings which are collected in real time.
